# Totis
-nombre del proyecto
-Descripcion
Actualmente existe una falta de información y orientación para los estudiantes de nuevo ingreso, especialmente para aquellos que tienen poco o ningún conocimiento previo en programación.
Muchos comienzan sus estudios sin comprender conceptos básicos, herramientas, metodologías o el campo laboral relacionado con la programación, lo que puede generar confusión, desmotivación, bajo rendimiento académico e incluso abandono temprano.
-Público objetivo
	•	Principalmente: estudiantes de nuevo ingreso.
	•	Secundariamente: personas de la comunidad interesadas en aprender programación.
